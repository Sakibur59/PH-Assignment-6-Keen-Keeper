> **Stay keen. Stay connected.**

## 🚀 Features

| Feature | Description |
|--------|-------------|
| 📞 Call Tracking | Log call activities with friends |
| 💬 Text Tracking | Record text message interactions |
| 🎥 Video Tracking | Track video call sessions |
| 🕒 Unified Timeline | Real-time activity feed with timestamps |
| 🔍 Filter System | Filter timeline by Call / Text / Video |
| 📊 Pie Chart Analytics | Visual breakdown of activity distribution |
| 🔔 Toast Notifications | Instant feedback on every action |
| ⚡ Responsive UI | Clean and mobile-friendly interface |

---

## 🛠️ Tech Stack

- **Frontend** — React JS ⚛️
- **State Management** — Context API
- **Routing** — React Router DOM
- **Charts** — Recharts 📊
- **Notifications** — React Toastify 🔔
- **Styling** — Tailwind CSS 🎨

---








## 📊 Timeline Filter

Filter your activity feed by:

- **All** — View every interaction
- **📞 Call** — Only call activities
- **💬 Text** — Only text activities
- **🎥 Video** — Only video activities

---

## 📈 Analytics — Pie Chart

The built-in pie chart gives you an instant visual summary:

- 🔵 Total Calls
- 🟢 Total Texts
- 🟡 Total Videos

Quickly understand how you're staying connected with the people who matter.
